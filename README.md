# Learning NLTK

This repository is for learning and experimenting with the **Natural Language Toolkit (NLTK)** in Python.

## Overview

NLTK is a popular library for natural language processing (NLP).  
It provides tools for:

- Tokenizing text (splitting sentences/words)
- Removing stopwords
- Lemmatization and stemming
- Part-of-speech tagging
- Named entity recognition
